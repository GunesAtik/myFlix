myFlix Project

An application which will provide informations about movies, directors and genres.

Users can sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all the movies.
Get data about a single movie by title /movies/:title GET> None A JSON object holding data about a single movie. Example:
{
id: 1,
title: "Reconstruction",
description: "Reconstruction is a psychological romantic drama film and the debut of Christoffer Boe",
genre: "Drama",
director: "Christoffer Boe",
image URL: ""
}
Return genre of movie by title /movies/genres/:title GET None {
title: "Reconstruction",
genre: "Drama"
}
Return director of movie by title /movies/directors/:title GET None {
director: "Christoffer Boe",
bio: "Christoffer Boe is a Danish film director and screenwriter.",
birth year: 1974
}
Allow new users to register /users POST JSON {
username: "GAtik",
name: "Günes Atik",
email: "guenes.atik@gmail.com"
}
Allow users to update their user info /users/:username POST JSON {
username: "GAtik",
name: "Günes Atik",
email: "guenes.atik@gmail.com"
}
Allow users to add a movie to their list of favourites /users/:username/:movies/ PUT None Sucessful PUT request adding film 'MOVIE NAME' to user 'USERNAME'
Allow users to remove a movie from their favourites /users/:username/:movies DELETE None Sucessful DELETE request, deleting film 'MOVIE NAME' from user 'USERNAME's favourite list
Allow users to deregister /users/:username DELETE None Account deleted for 'USERNAME'
